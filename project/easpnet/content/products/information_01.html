網管與網路加值服務

數位通國際網路主機代管內容應用，領先業界推出四大應用管理服務。

數位管家

數位通國際推出最新網管解決方案數位管家，一指搞定所有網路問題。

企業面對市場環境激烈競爭，如何將有限資源整合集中用於核心範疇增力企業競爭力，是每位企業主急於解決的問題。於是資訊委外( IT Outsource )服務的興起正是解決這令企業頭痛問題的特效藥。企業可以將內部非核心事業委外，達成將「專業的事交給專業的人處理」，並全心專注於核心事業的經營。

因此數位通國際所推出的「數位管家，一指搞定」服務，便是針對此一狀況推出的專案服務，期望一舉解決企業對內部網管監控服務的困擾，並能節省人、物力的成本支出，倍增企業競爭力。

數位管家服務內容：
  • PC設備管理服務 (Personal Computer Management Service)
  • 伺服器系統管理服務 (Server OS System Management Service)
  • 網路管理服務 (Network Management Service)
  • 應用伺服器管理服務 (Application Server Management Service)
  • 資料庫管理服務 (DB Server Management Service)
  • Security 安全管理服務 (SePSS Power Supply Service)
  • Monitoring 網路監控服務 (7*24 Monitoring Management Service)

網路服務

數位通國際推出最新網路服務，從規劃到建置一次完成。

企業的競爭力，已愈來愈依賴有線和無線網路資源。而企業最關切的，莫過於網路的完整性、可用性、可擴充性和技能。數位通國際專業技術工程團隊透過主動監控和管理來提升網路效能和可用性，回應動態的業務需求，降低總擁有成本。使用業經實證且安全的工具和方法來管理網路資源，提供彈性的開放式解決方案，迅速獲得網路投資的潛在報酬。我們將能協助您瞭解各種現有技術的特性及其間的差異，以便您根據特定業務需求及量化業務目標，來挑選裝置、建置解決方案，更有助於加速新網路技術的建置，爭取成本和競爭優勢。

網路服務內容：
  • 設備安裝建置
  • 網路接線
  • 基本管理
  • 網路設備 OS版本更新服務
  • 定期檢查
  • 新增或更換模組及記憶體
  • 網路連線問題診斷
  • 硬體維護

防火牆

數位通國際推出最新防火牆服務，安全萬無一失滴水不漏

企業內容資料安全性進幾年來日驅嚴重，網路上不同攻擊方式手法與企業內部機密資料外洩等問題日趨嚴重，避免有問題的封包影響到內部主機的安全，企業主不得須特別重視所帶來的災害與資安風險管理。而防火牆 (Firewall) 就是各企業及組織在建置資安控管解決方案當中最常被優先考量的安全控管機制。目前防火牆的運作參數之複雜程度也相對提高，然而除了這些技術層面的問題之外，當企業或組織大部分都已經建置了防火牆之後，接下來要如何更進一步做好管理工作，以強固防火牆整體的安全控管機能。

數位通國際專業技術服務團隊提供企業主強化內部網路安全性，及提高網路/人員工作效率，將為企業主節省大量成本及人力支出。

  • 安裝建置-提供 Internet 與 Intranet連結設定
    • 配合客戶需求完成各項 Rule 設定
  • 基本管理-組態檔代管-任何異動均保留一份最新組態檔
    • Rule 修改
    • 技術諮詢服務
  • 帳號管理
  • 版本更新服務-客戶須提供新版之 OS 供更新使用
    • 備份舊有組態檔
  • 硬體設備服務-須另外簽訂不同等級設備維護合約

系統服務

數位通國際推出最新系統服務，協調溝通加速軟體運作效能。

數位通國際網路除了提供主機代管網路連線服務之外，提供企業主其相關硬體設施之作業系統(OS)與資料系統(DB)整合管理服務，如資料庫備份(Windows/Linus/Oracle)透過數位通國際完整軟硬體設施及認證技術工程師，提供企業之重要資料文件免除意外發生而造成損失。而對企業代管硬體設施部份(Server/Firewall)可代為操作與維護管理，並提供系統效能分析使其網路效能發揮之最佳狀態。

系統服務內容：
  • 系統基本管理
  • 作業系統安裝與設定
  • 版本升級
  • 修補程式安裝
  • 套裝軟體安裝
  • 定期檢查
  • 系統安全健診
  • 系統效能檢測
  • 備份服務
  • 硬體維護
  • 設備上架
